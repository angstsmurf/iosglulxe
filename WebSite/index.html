Quick start

Already know how IF works? Here’s the scoop for this app:

  • You’ve got four tabs: the game, a note-taking journal, this help screen, and app preferences.
  • Swipe left or right to switch tabs.
  • Double-tap any word on the screen to copy-and-paste it into your command.
  • Tap the bookmark input button to bring up command shortcuts. There’s a palette of common IF commands, and a list of the commands you’ve recently used.
  • Font and theme preferences are in the menu in the top left.
  • VoiceOver output and voice dictation input are fully supported (on devices that offer them). Try them!

How to play

In this game, you are the center of a story. You will type commands in simple English; the game will respond, and tell you what happens.

What commands does this game understand? Well, it depends on the game. You can try nearly anything you can think of, if it makes sense in the situation you find yourself in.

Most interactive fiction follows some simple rules, though. Once you’re familiar with the pattern and the common commands, you can explore freely, and you’ll have a good idea how to experiment further.

What commands look like

The simplest IF commands are usually one or two words:

  • LOOK
  • TAKE SWORD
  • EAT CUPCAKE
  • PUSH BUTTON

But sometimes you need more:

  • LOOK UNDER BED
  • PUT LETTER IN ENVELOPE

In a more complicated scene, you might have to get more specific:

  • EAT THE SMALL BLUE JELLYBEAN
  • PUT COPPER COIN IN SLOT 4

You will almost always be dealing with concrete things that you see in the game world. Work with what you see around you. If you try wild abstractions like “GO SEE A MOVIE” or “RUN FOR PRESIDENT”, the game will probably not understand you.

Orienting yourself

If you are completely at sea, you can always try these commands:

  • LOOK or L : Look around the room — repeat the description of everything you see.
  • INVENTORY or I : List everything you’re carrying.
  • EXAMINE thing or X thing : Look more closely at something — learn more about it.

Because these actions are so common, they have one-letter abbreviations: “L”, “I”, and “X”.

Again, your actions should focus on the objects and exits that you see in the game world.

Moving around the world

The game world is divided into locations — usually called “rooms”, although they might be indoors or outdoors. A room is everything you can see at one time, and everything you can conveniently reach. Don’t worry about crossing the room, turning around, or standing near things; that sort of movement is automatic.

To move to a different location, you’ll usually type simple compass directions:

  • NORTH
  • SOUTH
  • EAST
  • WEST
  • NORTHEAST
  • SOUTHWEST
  • UP
  • DOWN

You can abbreviate these directions as “N”, “NE”, and so on; also “U” and “D”.

More specific commands sometimes make sense:

  • IN
  • OUT
  • ENTER BANK VAULT
  • GO THROUGH ARCHWAY
  • CLIMB LADDER

Common actions

You’re probably staring at your dictionary and shuddering in terror (or anticipation!) of the infinite possibilities. Sometimes IF works that way. More often, you’ll run through a small list of common commands — some of which you’ve already seen.

  • EXAMINE thing
  • GET thing, DROP thing
  • PUT this IN that, PUT this ON that
  • PUSH thing, PULL thing, TURN thing
  • OPEN thing, CLOSE thing
  • LOCK this WITH that, UNLOCK this WITH that
  • TOUCH thing, SMELL thing, LISTEN TO thing
  • EAT thing, DRINK thing
  • WEAR thing, REMOVE thing
  • SEARCH thing, LOOK IN thing, LOOK UNDER thing, LOOK BEHIND thing
  • ENTER thing, SIT ON thing, CLIMB thing
  • GET IN thing, GET OFF thing, GET OUT OF thing
  • BREAK thing, CUT thing, CRUSH thing
  • WAIT

Special commands

  • SAVE : Save your current game position.
  • RESTORE : Restore a saved game; return to the saved position.

You can save any number of positions; you’ll see a list when you type the “RESTORE” command. (Of course, when you launch the app you’ll be where you left off. You don’t need to “SAVE” for that. But it’s usually wise to save every once in a while, in case you make a mistake or want to try something different ways.)

  • UNDO : Take back your last move.
  • RESTART : Start the game over from the beginning.

These are handy, but not as flexible as “RESTORE” and “SAVE”.

  • SCRIPT : Begin saving a transcript of your game session.
  • SCRIPT OFF : Turn off “script” mode.

A transcript is a text log of your game. To see the list of saved transcripts, switch to the Notes tab and tap the Transcripts button in the top left.

  • ABOUT : Read a brief introduction to the game.
  • VERBOSE : See a detailed description of each room you enter, every time.
  • BRIEF : Turn off “verbose”.
